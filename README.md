
# Questify AI

[Deployed Here](https://www.questify.life/)

### Description

Questify is a text-based choose your own adventure game that uses the Hume and GPT-4 API to offer players an immersive and limitless gaming experience.

## Features

- Mobile UI Friendly
- Unlimited Gameplay


## Tech Stack

- JavaScript
- NextJs
- TailwindCSS
- OpenAI API
- Hume API
- Vercel (Deployment)
